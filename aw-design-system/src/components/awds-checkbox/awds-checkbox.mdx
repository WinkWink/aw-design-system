# Checkbox

The `awds-checkbox` component is a token-driven form control.

## Token usage

This component consumes Figma-exported CSS variables:

- `--action-primary`
- `--surface-default`
- `--border-default`
- `--text-default`
- `--focus-ring`
- `--space-8`
- `--radius-sm`
