# Modal

The `awds-modal` component is a token-driven dialog used to display focused content or actions above the page.

## Token usage

This component consumes Figma-exported CSS variables:

- `--surface-default`
- `--surface-overlay`
- `--text-default`
- `--border-default`
- `--focus-ring`
- `--space-16`
- `--space-24`
- `--radius-lg`
- `--shadow-lg`
