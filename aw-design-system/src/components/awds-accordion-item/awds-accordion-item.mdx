# Accordion Item

The `awds-accordion-item` component is a token-driven expandable section used inside `awds-accordion`.

## Token usage

This component consumes Figma-exported CSS variables:

- `--surface-default`
- `--surface-subtle`
- `--text-default`
- `--text-muted`
- `--border-default`
- `--focus-ring`
- `--space-12`
- `--space-16`
- `--radius-md`
