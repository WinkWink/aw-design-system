# Accordion

The `awds-accordion` component is a token-driven container used to group expandable accordion items.

## Token usage

This component consumes Figma-exported CSS variables:

- `--surface-default`
- `--text-default`
- `--border-default`
- `--focus-ring`
- `--space-12`
- `--space-16`
- `--radius-md`
