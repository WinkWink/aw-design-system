# Button

The `awds-button` component is a token-driven button built with Stencil.

## Token usage

This component consumes Figma-exported CSS variables:

- `--action-primary`
- `--action-primary-hover`
- `--surface-default`
- `--focus-ring`
- `--space-12`
- `--space-16`
- `--radius-md`
